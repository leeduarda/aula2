Comandos Git

Inicialização das Atividades:

  • git fetch --all

    sincroniza o repositório local com o repositório remoto.

  • git pull

    faz o download das atualizações do servidor remoto.

Finalização das Atividades:

  • git status

    lista as modificações.

  • git add .

    adiciona as modificações no servidor local.

  • git commit -m "[texto]"

    monta as modificações para envio ao servidor remoto.

  • git push

    faz upload das modificações do servidor local para o servidor remoto.
